# startups.org.nz — feedback & issues

This is the public issue tracker for **[startups.org.nz](https://startups.org.nz)**, a community board for the New Zealand startup community. The application code lives in a separate private repo; this repo is just for **reporting bugs and requesting features**.

We're early and testing as we go, so thanks for bearing with us while we smooth things out.

- 🐞 [Report a bug](https://github.com/flintarc/startups.org.nz-public/issues/new?template=bug_report.yml)
- 💡 [Suggest a feature or idea](https://github.com/flintarc/startups.org.nz-public/issues/new?template=feature_request.yml)
- ✉️ Prefer email, or not on GitHub? **hello@startups.org.nz**
